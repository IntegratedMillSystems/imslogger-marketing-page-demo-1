The Project

IMS Logger was designed to make logging a system of PLCs as easy as adding a new switch. It includes:

  • A Producer/Consumer Connection, which tells the PLCs to send data at consistant time intervals (up to 1 millisecond).
  • Tag Detection and Tag Structure Detection
  • Live Tag View
  • Threaded Logging

Special thanks to the libraries which made this possible, pycomm3 and a private branch of pylogix.
